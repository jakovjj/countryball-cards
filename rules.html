🚀 Crash Course - Learn in 2 Minutes!

1. Setup

Each player draws 5 cards, reveals 3 face-up. Set up the resource market with random prices.

2. Your Turn

Get 5 coins → Buy/sell resources → Use abilities (pay resource costs) → Attack enemies

3. Combat

Roll D20, need the number shown next to ability name or higher to hit. Hit = stun enemy + 1 skull marker

4. Win

Give each enemy countryball 3 skulls to eliminate them. Last player standing wins!

🌍 Game Overview

1v1 tactical card game where you control nations in strategic combat. Deploy countryballs, manage resources, and eliminate your opponent's forces.

Players: 2
Time: 30-45 min
Age: 12+

📦 Components

Countryball Cards (nations with unique abilities) • D20 Die (combat rolls) • Resources (🌾 Grains, 🐎 Horse, 🛢️ Oil, ⚙️ Iron) • Skull Markers (track damage) • Coins (currency)

⚙️ Setup

1. Draw 5 cards each (keep hidden) → 2. Reveal 3 cards face-up (your army) → 3. Setup market with 4 resources at random prices → 4. First player: who revealed cards first

🎮 How to Play

Players alternate turns with these phases:

1. Gain Coins & Trade

Get 5 coins, then buy/sell resources at market prices (2-5 coins each, prices change every round). Resources: Grains, Horse, Oil, Iron

2. Use Abilities

Pay resource costs to activate any abilities multiple times

3. Combat

Attack enemies: Pay resource cost → Roll D20 → Need number shown next to ability name or higher to hit → Hit = stun + 1 skull. Give each enemy countryball 3 skulls to eliminate them.

Note: Failed attacks still cost resources. Stunned cards can't act next turn.

🏆 Win Condition

Eliminate all enemy countryballs (3 skulls each) to win!

⚔️ Combat & Abilities

📜 Understanding Abilities

Format: Ability Name + Resource Cost (e.g. 🛢️🛢️ = 2 Oil) + Effect Description + Dice Roll (e.g. 🎲>=12 = need 12+ on D20)

⚔️ Attack Abilities

Some abilities are attacks that deal damage. Pay resources → If needed roll D20 → Need the number shown or higher → Success = by default stun target + 1 skull marker

🔮 Non-Attack Abilities

Some abilities provide other effects (healing, cresource generation, etc.) but may still require dice rolls for success

🔄 Stun Mechanic

Stunned cards can't use abilities next turn, then automatically reactivate

🎲 Multiple Targets

When an ability affects multiple targets, roll separately for each one

💀 Failed Attempts

Resources are still consumed even if the dice roll fails
